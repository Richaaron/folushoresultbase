
### 2️⃣ Test on Your Phone (2 minutes)

```bash
# Find your computer's IP:
Windows:  ipconfig  (look for IPv4 Address)
Mac:      ipconfig getifaddr en0
Linux:    hostname -I

# On your phone, visit:
http://YOUR_IP:5173

# (Keep app running in terminal!)
```

## 💡 Pro Tips

1. **Test in Browser First** - Use DevTools mobile view (faster iteration)
2. **Test on Real Device** - iOS and Android behave differently
3. **Clear Cache** - Press Ctrl+Shift+Delete if seeing old version
4. **Check Console** - Press F12 → Console to see any errors
5. **Use Landscape Too** - Test portrait AND landscape orientation

---

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Menu won't open | Refresh page (F5) |
| Text too small | Check browser zoom (Ctrl+0) |
| Table cut off | Swipe horizontally to scroll |
| Buttons too small | Already fixed! |
| Forms hard to fill | Try landscape orientation |
